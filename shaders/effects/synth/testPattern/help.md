# Test Pattern

NxN numbered checkerboard for identifying axis flips.

## Description

Generates a numbered grid pattern useful for debugging and testing texture coordinate orientation. Each cell is numbered to help identify axis flips or coordinate issues.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| gridSize | int | 4 | 1–16 | Number of grid cells per axis |
