# scope

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| color | color | [0, 1, 0] | - | Waveform color |
| thickness | float | 2 | 0.5–10 | Line thickness |
| gain | float | 1 | 0.1–5 | Amplitude gain |
