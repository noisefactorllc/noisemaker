# Solid

Fills the output with a solid color.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| r | float | 0.5 | 0–1 | Red channel value |
| g | float | 0.5 | 0–1 | Green channel value |
| b | float | 0.5 | 0–1 | Blue channel value |
