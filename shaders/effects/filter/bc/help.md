# Brightness/Contrast

Adjusts image brightness and contrast.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| brightness | float | 1 | 0–10 | Multiplies pixel values. Values < 1 darken, values > 1 brighten |
| contrast | float | 0.5 | 0–1 | Controls contrast. 0 = no contrast, 0.5 = normal, 1 = maximum |
