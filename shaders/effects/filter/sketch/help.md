# sketch

Pencil sketch with crosshatch shading

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| contrast | float | 2.0 | 0-3 | Sketch contrast |
| hatchDensity | float | 1.0 | 0-2 | Crosshatch line density |
| alpha | float | 1.0 | 0-1 | Effect opacity |
