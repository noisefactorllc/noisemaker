## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| x | float | 3 | 1–20 | Horizontal repeat factor |
| y | float | 3 | 1–20 | Vertical repeat factor |
| offsetX | float | 0 | -1–1 | Horizontal offset |
| offsetY | float | 0 | -1–1 | Vertical offset |
| wrap | int | 1 (repeat) | mirror/repeat/clamp | Edge wrapping mode |
