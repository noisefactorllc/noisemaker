# degauss

CRT degauss effect

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| displacement | float | 0.0625 | 0-0.25 | Displacement amount |
| speed | float | 1.0 | 0-2 | Animation speed |
| seed | int | 1 | 1-100 | Random seed |
