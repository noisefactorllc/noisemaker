# tunnel

Perspective tunnel effect with shape options

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| shape | int | circle | circle/triangle/square/hexagon/octagon | Shape |
| scale | float | 1 | 1-3 | Scale |
| speed | int | 1 | -5-5 | Forward Speed |
| rot speed | int | 0 | -2-2 | Rotation speed |
| 1:1 aspect | boolean | true | - | 1:1 Aspect |
