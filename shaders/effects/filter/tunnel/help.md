## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| shape | int | 0 | circle, triangle, square, hexagon, octagon | Tunnel cross-section shape |
| speed | float | 1 | -5–5 | Animation speed through tunnel |
| rotation | float | 0 | -2–2 | Tunnel rotation |
| scale | float | 0 | -2–2 | Tunnel scale |
