# Tunnel

Creates a tunnel effect with configurable cross-section shape.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| shape | int | circle | circle/triangle/square/hexagon/octagon | Tunnel cross-section shape |
| scale | float | 0 | -2–2 | Tunnel scale |
| forwardSpeed | float | 1 | -5–5 | Animation speed through tunnel |
| rotationSpeed | float | 0 | -2–2 | Tunnel rotation speed |
