# lowPoly

Low-polygon style render using Voronoi cells

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| scale | int | 50 | 2-100 | Cell size |
| seed | int | 1 | 1-100 | Random seed |
| alpha | float | 1.0 | 0-1 | Opacity |
