# chroma

Isolate specific hue range with feathering

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| targetHue | float | 0.33 | 0-1 | Target Hue |
| range | float | 0.1 | 0-0.5 | Range |
| feather | float | 0.05 | 0-0.25 | Feather |
