# Tint

Multiplies the image by RGB color values.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| r | float | 1 | 0–1 | Red channel multiplier |
| g | float | 1 | 0–1 | Green channel multiplier |
| b | float | 1 | 0–1 | Blue channel multiplier |
