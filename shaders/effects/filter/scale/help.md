# scale

| scale y | float | 0.5 | 0-10 | Vertical scale |
| centerX | float | 0.5 | 0-1 | Horizontal center |
| centerY | float | 0.5 | 0-1 | Vertical center |
| wrap | int | repeat | mirror/repeat/clamp | Wrap |
