# glowingEdge

Glowing edge detection

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| shape | int | circle | circle/diamond/square/star | Edge shape |
| width | int | 1 | 0-10 | Edge width in texels |
| alpha | float | 1 | 0-1 | Opacity |
