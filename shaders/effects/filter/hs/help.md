# hs

Adjust hue and/or saturation

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| rotation | float | 0 | -180-180 | Hue Rotation |
| hueRange | float | 100 | 0-200 | Hue Range |
| saturation | float | 1 | 0-4 | Saturation |
