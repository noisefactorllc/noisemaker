# Translate

Translates (moves) the image horizontally and vertically.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| x | float | 0 | -1–1 | Horizontal translation |
| y | float | 0 | -1–1 | Vertical translation |
| wrap | int | repeat | mirror/repeat/clamp | Edge wrapping mode |
