# Palette

Applies a color palette lookup to the image.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| paletteIndex | int | afterimage | palette enum | Palette to apply |
| paletteRotation | int | none | none/fwd/back | Palette rotation direction |
| paletteOffset | float | 0 | 0–100 | Palette offset |
| paletteRepeat | float | 1 | 0.1–10 | Palette repeat factor |
| alpha | float | 1.0 | 0–1 | Blend amount with original |
