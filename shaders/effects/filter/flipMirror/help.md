# FlipMirror Effect

Flip and mirror image transformations. Provides horizontal/vertical flipping and various mirroring modes.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| mode | int | none | none/all/horizontal/vertical/mirrorLtoR/mirrorRtoL/mirrorUtoD/mirrorDtoU/mirrorLtoRUtoD/mirrorLtoRDtoU/mirrorRtoLUtoD/mirrorRtoLDtoU | Flip/mirror mode |

## Modes

- **none**: No transformation
- **all**: Flip both horizontal and vertical
- **horizontal**: Flip horizontally only
- **vertical**: Flip vertically only
- **mirrorLtoR**: Mirror left side to right
- **mirrorRtoL**: Mirror right side to left
- **mirrorUtoD**: Mirror top to bottom
- **mirrorDtoU**: Mirror bottom to top
- **mirrorLtoRUtoD**: Mirror left-to-right and up-to-down
- **mirrorLtoRDtoU**: Mirror left-to-right and down-to-up
- **mirrorRtoLUtoD**: Mirror right-to-left and up-to-down
- **mirrorRtoLDtoU**: Mirror right-to-left and down-to-up
