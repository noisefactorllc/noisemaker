## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| strength | float | 25 | 0–100 | Controls the intensity of the pinch effect |
| aspectLens | boolean | true | — | When enabled, applies pinch in a circular region regardless of canvas aspect ratio |
| wrap | int | 0 (mirror) | mirror/repeat/clamp | Edge wrapping mode |
| rotation | float | 0 | -180–180 | Rotation in degrees |
