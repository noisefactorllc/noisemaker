# voronoi

Voronoi cell pattern

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| diagramType | integer | 1 | 0-7 | Diagram Type |
| nth | integer | 0 | -4-4 | Nth Neighbor |
| shape | integer | 1 | 1-4 | Shape |
| sdfSides | integer | 3 | 3-24 | SDF Sides |
| alpha | float | 1 | 0-1 | Alpha |
| withRefract | float | 0 | 0-2 | Refract |
| inverse | boolean | false | - | Inverse |
| ridgesHint | boolean | false | - | Ridges Hint |
| refractYFromOffset | boolean | true | - | Refract Offset |
| pointFreq | integer | 3 | 1-10 | Point Frequency |
| pointGenerations | integer | 1 | 1-5 | Generations |
| pointDistrib | integer | 0 | 0-9 | Distribution |
| pointDrift | float | 0 | 0-1 | Point Drift |
| pointCorners | boolean | false | - | Include Corners |
| downsample | boolean | true | - | Downsample |
