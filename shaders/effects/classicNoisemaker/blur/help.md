# Blur

Applies a blur effect to the image.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| Amount | float | — | — | Downsample factor that controls blur radius |
| Spline Order | int | — | 0–3 | Interpolation order (0=Const / 1=Linear / 2=Cosine / 3=Bicubic) |
