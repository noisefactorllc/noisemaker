## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| volumeSize | int | x64 | x16, x32, x64, x128 | Volume resolution |
| fractalType | int | mandelbulb | mandelbulb, mandelcube, juliaBulb, juliaCube | Fractal type |
| power | float | 8 | 2–16 | Fractal power exponent |
| iterations | int | 10 | 1–20 | Maximum iterations |
| bailout | float | 2 | 1–8 | Escape radius |
| juliaX | float | 0 | -100–100 | Julia constant X |
| juliaY | float | 0 | -100–100 | Julia constant Y |
| juliaZ | float | 0 | -100–100 | Julia constant Z |
| colorMode | int | mono | mono, orbitTrap, iteration | Color mode |
| seed | float | 0 | 0–100 | Random seed |
