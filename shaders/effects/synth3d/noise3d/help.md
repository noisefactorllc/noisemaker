# noise3d

3D simplex noise volume

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| volumeSize | int | x64 | x16/x32/x64/x128 | Volume size |
| scale | float | 3 | 0-100 | - |
| octaves | int | 1 | 1-6 | - |
| colorMode | int | mono | mono/rgb | Color mode |
| ridges | boolean | false | - | - |
| seed | int | 0 | 0-100 | - |
| speed | int | 1 | 0-5 | - |
