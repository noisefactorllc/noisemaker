# Cell Refract

Cell-based refraction effect.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| shape | int | 1 | circle/diamond/hexagon/octagon/square/triangle | Cell shape metric |
| scale | float | 50 | 1–100 | Overall scale |
| cellScale | float | 75 | 1–100 | Individual cell scale |
| cellSmooth | float | 0 | 0–100 | Cell edge smoothing |
| cellVariation | float | 0 | 0–100 | Cell position variation |
| loopAmp | int | 1 | 0–5 | Animation speed |
| kernel | int | 0 | none/blur/derivatives/derivDivide/edge/emboss/litEdge/outline/pixels/posterize | Post-processing kernel |
