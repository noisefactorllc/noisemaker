# Splat

Splatter paint effect with configurable splats and specks.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| useSplats | boolean | true | — | Enable splat effect |
| useSpecks | boolean | true | — | Enable speck effect |
| splatScale | float | 3 | 1–5 | Scale of splat shapes |
| splatCutoff | float | 25 | 0–100 | Splat visibility threshold |
| splatSpeed | int | 1 | 0–5 | Animation speed for splats |
| splatSeed | int | 1 | 1–100 | Random seed for splat pattern |
| splatMode | int | 2 | color/displace/invert/negative | Splat rendering mode |
| splatColor | vec3 | [1,1,1] | — | Color for splats |
| speckScale | float | — | — | Scale of speck shapes |
