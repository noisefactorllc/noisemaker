# splat

Splatter paint effect

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| useSplats | boolean | true | - | Splats |
| useSpecks | boolean | true | - | Specks |
| splatScale | float | 3 | 1-5 | Splat scale |
| splatCutoff | float | 25 | 0-100 | Splat cutoff |
| splatSpeed | int | 1 | 0-5 | Splat speed |
| splatSeed | int | 1 | 1-100 | Splat seed |
| splatMode | int | invert | color/displace/invert/negative | Splat mode |
| splatColor | vec3 | 1,1,1 | - | Splat color |
| speckScale | float | 5 | 1-5 | Speck scale |
| speckCutoff | float | 70 | 0-100 | Speck cutoff |
| speckSpeed | int | 1 | 0-5 | Speck speed |
| speckSeed | int | 1 | 1-100 | Speck seed |
| speckMode | int | color | color/displace/invert/negative | Speck mode |
| speckColor | vec3 | 0.8,0.8,0.8 | - | Speck color |
