# Palette

Color palette mapping effect.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| paletteType | int | 0 | cosine/fiveColor | Palette generation method |
| cyclePalette | int | 1 | off/forward/backward | Animate palette cycling |
| rotatePalette | float | 0 | 0–100 | Manual palette rotation offset |
| freq | int | 1 | 1–4 | Color frequency multiplier |
| color1 | vec3 | [1,0,0] | — | First color (five-color mode) |
| color2 | vec3 | [1,1,0] | — | Second color (five-color mode) |
| color3 | vec3 | [0,1,0] | — | Third color (five-color mode) |
| color4 | vec3 | [0,1,1] | — | Fourth color (five-color mode) |
| color5 | vec3 | — | — | Fifth color (five-color mode) |
