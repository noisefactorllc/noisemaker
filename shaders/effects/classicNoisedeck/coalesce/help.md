# Coalesce

Coalescing blend effect with multiple blend modes.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| tex | surface | none | — | Source surface B |
| seed | int | 1 | 1–100 | Random seed |
| blendMode | int | 10 | add/alpha/brightnessAB/brightnessBA/cloak/colorBurn/colorDodge/darken/difference/exclusion/glow/hardLight/hueAB/hueBA/lighten/mix/multiply/negation/overlay/phoenix/reflect/saturationAB/saturationBA/screen/softLight/subtract | Blend mode |
| mixAmt | float | 0 | -100–100 | Mix amount |
| refractAAmt | float | 0 | 0–100 | Refraction A to B |
| refractBAmt | float | 0 | 0–100 | Refraction B to A |
