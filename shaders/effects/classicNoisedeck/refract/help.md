# Refract

Refraction distortion effect.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| seed | int | 1 | 1–100 | Random seed |
| blendMode | int | 10 | add/colorBurn/colorDodge/darken/difference/exclusion/glow/hardLight/lighten/mix/multiply/negation/overlay/phoenix/reflect/screen/softLight/subtract | Blend mode |
| mixAmt | float | 50 | 0–100 | Mix amount |
| mode | int | 0 | refract/reflect | Distortion mode |
| amount | float | 50 | 0–100 | Distortion amount |
| refractDir | float | 0 | 0–360 | Refraction direction in degrees |
| wrap | int | 0 | clamp/mirror/repeat | Edge wrap mode |
