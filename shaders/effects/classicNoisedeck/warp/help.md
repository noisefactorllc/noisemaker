# Warp

Warp distortion effect with multiple distortion types.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| distortionType | int | 10 | bulge/perlin/pinch/polar/spiralCw/spiralCcw/vortex/waves | Distortion type |
| flip | int | 0 | none/all/horizontal/vertical/leftToRight/rightToLeft/upToDown/downToUp/lrUd/lrDu/rlUd/rlDu | Flip or mirror mode |
| scale | float | 1 | -5–5 | Distortion scale |
| rotateAmt | float | 0 | -180–180 | Rotation in degrees |
| strength | float | 25 | 0–100 | Distortion strength |
| seed | int | 1 | 1–100 | Noise seed |
