# Shapes

Geometric shapes generator with dual animation loops.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| loopAOffset | int | 40 | circle/triangle/diamond/square/.../rings/sine | Loop A shape |
| loopBOffset | int | 30 | circle/triangle/diamond/square/.../rings/sine | Loop B shape |
| loopAScale | float | 1 | 1–100 | Loop A scale |
| loopBScale | float | 1 | 1–100 | Loop B scale |
| loopAAmp | float | 50 | -100–100 | Loop A amplitude |
| loopBAmp | float | — | -100–100 | Loop B amplitude |
| wrap | boolean | — | — | Enable seamless wrapping |
| paletteMode | int | — | — | Palette mode |
| cyclePalette | int | — | — | Palette cycling |
| rotatePalette | float | — | — | Palette rotation |
| repeatPalette | int | — | — | Palette repetitions |
| seed | int | — | 1–100 | Random seed |
