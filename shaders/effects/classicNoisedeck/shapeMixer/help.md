# Shape Mixer

Shape-based mixing effect with palette support.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| blendMode | int | 2 | add/divide/max/min/mix/mod/multiply/reflect/refract/subtract | Blend mode |
| loopOffset | int | 10 | none/circle/triangle/diamond/square/.../rings/sine | Animation loop shape |
| loopScale | float | 80 | 1–100 | Shape scale |
| animate | int | 1 | off/forward/backward | Animation direction |
| palette | int | 41 | — | Color palette |
| paletteMode | int | 0 | — | Palette mode |
| cyclePalette | int | — | — | Palette cycling |
| rotatePalette | float | — | — | Palette rotation |
| repeatPalette | int | — | — | Palette repetitions |
