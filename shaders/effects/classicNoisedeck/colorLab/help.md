# Color Lab

Color manipulation lab with multiple color spaces and palettes.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| seed | int | 1 | 1–100 | Random seed |
| colorMode | int | 2 | mono/linearRgb/srgbDefault/oklab/palette | Color space mode |
| palette | int | 46 | — | Color palette |
| paletteMode | int | 0 | — | Palette mode |
| cyclePalette | int | 1 | off/forward/backward | Palette cycling |
| rotatePalette | float | 0 | 0–100 | Palette rotation |
| repeatPalette | int | 1 | 1–5 | Palette repetitions |
| hueRotation | float | 0 | 0–360 | Hue rotation offset |
