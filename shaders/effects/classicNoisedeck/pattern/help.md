# Pattern

Pattern generator with various geometric patterns.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| patternType | int | 1 | checkers/dots/grid/hearts/hexagons/rings/squares/stripes/waves/zigzag/truchetLines/truchetCurves | Pattern type |
| scale | float | 80 | 1–100 | Pattern scale |
| skewAmt | float | 0 | -100–100 | Skew amount |
| rotation | int | 0 | -180–180 | Rotation in degrees |
| lineWidth | float | 100 | 1–100 | Line or element width |
| seed | int | 1 | 1–100 | Random seed |
| color1 | vec3 | — | — | Primary color |
| color2 | vec3 | — | — | Secondary color |
