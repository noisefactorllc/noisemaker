# Text

Text overlay renderer.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| seed | int | 1 | 1–100 | Random seed |
| glyphUV1 | vec2 | [0,0] | — | Glyph UV start coordinates |
| glyphUV2 | vec2 | [1,1] | — | Glyph UV end coordinates |
| font | int | 4 | cursive/fantasy/monospace/nunito/sansSerif/serif/vcr | Font family |
| text | int | 0 | — | Text content selector |
| position | int | 4 | topLeft/topCenter/topRight/midLeft/midCenter/midRight/bottomLeft/bottomCenter/bottomRight | Text position |
| color | vec3 | [1,1,1] | — | Text color |
